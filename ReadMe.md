# BTB.

A tool by Amlal EL Mahrouss to build libraries and programs.

## Installation requirements:

- GNU C++ Compiler.

## Steps (makefile):

- Run make `build-btb-windows` or `build-btb` for POSIX.

## Steps (btb):

- Run `btb` and pass the path to the manifest file.
